Try-me example: MAE (arXiv 2111.06377)
PaperLens was trained on anonymized inputs, so drop in a paper PDF whose authors / affiliations are blinded. To grab MAE for a quick try: 
curl -L https://arxiv.org/pdf/2111.06377 -o mae.pdf
Note this is the de-anonymized arXiv copy — it'll still score, but you'll get a calibrated number by passing an anonymized PDF (or use the LaTeX source / arXiv link tabs, which run paperprep's anonymization first).

Try-me example: MAE (arXiv 2111.06377)
Fetch the LaTeX source bundle from arXiv and extract it, then point Browse at the resulting dir: 
mkdir -p ~/mae_src
curl -L https://arxiv.org/src/2111.06377 | tar xz -C ~/mae_src
Click Browse… above, navigate to ~/mae_src, and hit ✓ Use this folder. paperprep anonymizes the source (authors → Anonymous, strips URLs/ack/thanks) before compile + score.

If the dir is a git repo, an extra History option appears — score across commits to see how p_accept evolved over your paper's lifetime.
